回旋镖

最近同事遇到一个 bug，看描述是 debug 了挺久才定位到问题。

我一看，乐了，这个 bug 在我刚入职的时候遇到了。

这个 bug 是由于某个 API 的入参默认设置得有问题，不符合预期效果。

偏偏这个 bug 触发的几率很诡异。

之所以这么诡异，是因为某个 API 手动敲的人少，日常使用都是复制粘贴旧的代码完事。

每次手动敲的时候，就可能缺省最后一个参数，从而导致泄漏。

这波啊，只能说是回旋镖打到了队友。

我当初也没重视，如果当初提修改，应该就可以让队友避免一些坑。

立心栽花花不香

中了一段时间的番茄和西瓜枯萎了。

总的来说，是生病之后，我喷药没把握好，浓度太高，导致烧苗死了。

心疼我的番茄西瓜，两张照片留念。

番茄虽然只有指甲大小，但是味道确实非常浓郁，就是美好的番茄味。

即使不吃，摆着看，翠绿中带有一些果实点缀，也是非常养眼。

西瓜因为我知道自己盆小，所以只留了一个，其他的都被我疏了。

这个西瓜刚处于膨果期，但是没发育完全，所以一半是圆滚滚的，另一半有点发育不够。

不过水分是充足的，并且很甜，完全符合一个西瓜的表现。

于是我刚又下单了番茄苗。

回头种到大盆里边试试。

大的要来了

最近法环 DLC 要来了，我却还要上班，可惜啊。

我已经迫不及待了。

这次决定试试法师流派。

带上我的暗月大剑。

其实这个还有周边来着。

太酷啦！

之前和朋友讨论过，如果真能送我就好，

我也并不是多想要啦，毕竟这玩意挺贵重的，还没啥卵用，

送我我也没地方放是吧？

只是说，如果能送我就好了（
