很帅的代码

最近看到一个很帅的代码，作为一名“脚本小子”，“API调用者”来说，确实很帅。

onUpdate(dt){
    const dis = dt;
    this.cube.position.y = Math.sin(dis) * 0.3;
}

这段代码定义了一个正弦运动，作为cube的上下缓动。

天冷加衣

最近广东经历了断崖式的降温。

挑了几套好看的外套，但是迟迟没下单，有点拖延症——即使心想自己穿起来肯定很帅。

晚上吃了饭，跑去天台站了一会。楼梯口遇到个吸烟的，真是晦气，还好天台的空气凌冽，让我精神抖擞。

我必须感谢这凌冽的空气，以及同事们的懒惰。导致天台一个人也没有，实在是太好了。

我足足在寒风中伫立了十几二十分钟，风吹得我脸部僵硬，手脚冰冷。

看着四周密封的高墙，纵身一跃的缝隙也没有。

天台上有几株植物，其中一株居然结了一个小柠檬，既有未开的花苞，也有正在绽放的花朵。

我忍不住拿出手机来拍了几张照片。

这个青柠檬，咬一口，肯定很酸，还没成熟。

陈年记录

下午同事们把室内的空调开为了制热模式，一开始有点儿用力过猛了，导致我感觉自己都快要窒息了。

后来我主动去调低了一点，其实也有不少同事受不了，可惜大伙都有点懒得动。

最近我也没啥动力上班，反正上班效率不高，下午忽然想起了以前的一个勉强算得上是记事本的软件账号。

扫码进去，看到了以前的一些记录。

各式各样的都有，有一些有趣的段子或者乱七八糟的评论。

以及，一些让我看得黯然神伤的记录。

大抵是过去的我留下的遗物吧，迟点有空整理整理，也就发上来。当是一种纪念好了。

当然，会修改到正确的日期，嗯，很多记录甚至在这个博客之前。

只能说，彼一时，此一时，此时此刻，虽然能从只言片语中看到一点点当初的心境，但究竟也是不一样，很多看法，想法，与当时不同，甚至大相径庭。

哈，当然里边也有一些黄色记录，看了会让人口干舌燥的那种。

实际这部分依旧会让我神伤，因为太多太多的美好，我不曾经历，未来也不会再有。
